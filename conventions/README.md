# Facility Conventions

This directory contains RepoConventions conventions shared by FacilityApi repositories.

## Conventions

- [facility-dotnet-package](./facility-dotnet-package/README.md) composes the standard conventions for FacilityApi .NET package repositories.
- [facility-dotnet-package-props](./facility-dotnet-package-props/README.md) manages universal FacilityApi package properties.
- [facility-dotnet-package-workflows](./facility-dotnet-package-workflows/README.md) manages the standard package CI, Copilot setup, and convention-application workflows.
